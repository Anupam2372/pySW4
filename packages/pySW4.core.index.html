Core routines of pySW4

Introduction¶

The core package provides the main post-processing routines of the pySW4 package. It has some scripts which can be run from the command line for quick plotting. It may be usefull to run these scrips on the server-end, even while the computation is running in order to generate pseudo-RunTime visualization of the results. See the scripts documentations for more information.
